# Sound Files

This directory contains sound effect files for the 2048 game:

- `move.mp3` - Played when tiles move
- `merge.mp3` - Played when tiles merge
- `gameover.mp3` - Played when the game ends

## Format
All sound files are in MP3 format for broad browser compatibility.
